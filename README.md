# Aegis - Frontend

## Descripción

**Aegis** es una aplicación móvil desarrollada con **React**, **Expo** y **TypeScript** para ofrecer funcionalidades de seguridad personal y localización en tiempo real. La app permite a los usuarios compartir su ubicación en vivo con amigos y familiares, garantizar su seguridad mediante alertas y acceder a funciones que mejoren su tranquilidad al estar en movimiento.

Este repositorio contiene el **frontend** de la aplicación, que está construido utilizando **React Native** con **Expo** y **TypeScript**.
